# invoices2
A better version full stack app MEAN of the invoicing application.

# Assumptions
The application is used to issue invoices online. 

# Server run:

Before run server please add your server/.env file like: <br />
`API_URL = ''` <br />
`CONNECTION_STRING = '' ` <br />
`PORT = '' `<br />
`SECRET = ''`
